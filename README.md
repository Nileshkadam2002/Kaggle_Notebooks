🚀 Featured Project: Global Billionaires Analysis

I am currently analyzing the Global Billionaires Dataset to understand the landscape of extreme wealth in 2024.

🔍 Key Insights Explored:
Wealth Distribution: Which countries and industries produce the most billionaires?

Demographics: Analysis of age, gender, and source of wealth (Self-made vs. Inherited).

Trends: Correlation between GDP and billionaire count per capita.
